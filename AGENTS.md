# Repository Instructions

Antes de qualquer trabalho neste repositorio:

1. Executar `git remote -v`, `git fetch --all --prune` e `git status -sb`
2. Informar ao usuario se existem atualizacoes remotas
3. Perguntar se ele deseja baixar o repositorio atualizado antes de continuar
4. Se o usuario aprovar, executar `git pull --ff-only`

Nunca salvar token em arquivos versionados ou na URL remota.
